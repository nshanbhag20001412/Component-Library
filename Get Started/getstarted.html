Quick Start

Style Link

To start with include the css link mentioned below inside your head tag.

<link rel="stylesheet" href="https://integral-ui-cl.netlify.app/CSS/allstyles.css">

Icons

Components like badge, buttons, cards, ratings has presence of some icons that have been derived from font awesome. To use them add the script below.

<script src="https://kit.fontawesome.com/0ce6b5a07c.js" crossorigin="anonymous"></script>

Theme

Colors

Integral UI uses some theme colors in the fom of color variable for there components. You can overwrite these color variables according to yourself.

Primary Color
var(--primary)
Off White
var( --off-white)
Error color
var(--error)
Warning Color
var(--warning)
Success color
var(--success)
Blue color
var(--blue)
Black color
var(--black)

Fonts

Integral UI uses Mate font throughout it's library. The font comes included with the css styling link. Still if you wish to have it seperately you can copy from below.

@import url('https://fonts.googleapis.com/css2?family=Mate&family=Vollkorn+SC:wght@600&display=swap');
