Alert

Alerts are important information prompts that needs to user attention. Here are few alerts you can make use of.

  • This is a default alert
  • This is a primary alert
  • This is a success alert
  • This is an error alert
  • This is a warning alert
  • This is Light color alert
  • This is Dark alert
    • Note: The icons used are from fontawesome. You need to add their link to use the icons from the code below

    //default alert
    <li class="list alert "> This is a default alert</li>

    //primary alert
    <li class="list alert alert-primary"> This is a primary alert</li>

    //Success alert
    <li class="list alert alert-success"><>i class="fas fa-check-square"><>/i> This is a success alert</li>

    //Error alert
    <li class="list alert alert-error"><i class="fas fa-exclamation-circle"></i> This is an error alert</li>

    //Warning alert
    <li class="list alert alert-warning"><i class="fas fa-exclamation-triangle"></i> This is a warning alert</li>

    //Light theme alert
    <li class="list alert alert-light">This is Light color alert</li>

    //dark theme alert
    <li class="list alert alert-dark"> This is Dark alert</li>
