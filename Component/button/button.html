Button

Solid Buttons

By default buttons will have theme color. You have the other colors from this library as below.

<button class="btn ">Button</button> //default button

//You can have the other colors from this library as follow
<button class="btn red">Button</button>

<button class="btn yellow">Button</button>

<button class="btn blue">Button</button>


Outline buttons

Similarly you can have outline variation of buttons

<button class="btn btn-outline">Button</button> //default outline button

//You can have the other colors from this library as follow
<button class="btn btn-outline-red">Button</button>

<button class="btn btn-outline-green">Button</button>

<button class="btn btn-outline-yellow">Button</button>

<button class="btn btn-outline-blue">Button</button>

Other variations

Here you can other variation of the buttons. To have the colours from the theme add repectively class to it, you can refer the solid buttons example for that.

Note: The icons used are from fontawesome. You need to add their link to use the icons from the code below

<button class="btn btn-link"><a href="#">Link</a></button> //button with link

<button class="btn "><i class="fas fa-user"></i></button> //icon button

<button class="btn btn-float"><i class="fas fa-plus"></i></button> //floating button

<button class=" btn btn-white">Button</button> //white button variation
