Input

Simple Input

By default the input has the theme color. According to you needs you can have these following input field when the field is in focus.

Success
Password is wrong
Need more characters
//Default
<div class="input-container">
<label for="input-field">Lorem</label>
<input class="input-field" type="text" placeholder="Start Typing">
</div>

//Success field
<div class="input-container ">
<label for="input-field ">Password</label>
<input class=" success input-field " type="text" placeholder="Start Typing">
<div class="input-field-status success "> Success </div>
</div>

//Error field
<div class="input-container ">
<label for="input-field">Password</label>
<input class=" error input-field " type="text" placeholder="Start Typing">
<div class="input-field-status error "> Password is wrong </div>
</div>

//Warning field
<div class="input-container ">
<label for="input-field ">Password
<input class="warning input-field " type="text" placeholder="Start Typing">
<div class="input-field-status warning "> Need more characters </div>
</div>
